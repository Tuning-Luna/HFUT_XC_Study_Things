## 关于

这个仓库建立是为了存放合肥工业大学宣城校区（HFUT-XC）计算机科学与技术（CS）大一到大四的学习资料（不仅仅包括自己的，也包括当时在网络上收集的，一并放在这里），同时也是为了方便学弟学妹们。
