    使用方式

    方式一：在线浏览（推荐）

    无需下载，直接访问在线网站查看全部资料：

    https://tuning-luna.github.io/HFUT_XC_Study_Things

    仓库与网站内容完全同步，在线找到所需文件后直接下载即可。

    方式二：完整克隆

    需要下载全部资料到本地时使用。

    前置条件：确保电脑已安装 Git

    git clone https://github.com/Tuning-Luna/HFUT_XC_Study_Things.git

    方式三：按需下载（仅下载部分文件）

    如果你只需要部分文件，推荐使用以下工具：

    GitZip 插件（Chrome / Edge）

    1. 安装 GitZip for GitHub
    2. 安装后打开本仓库 GitHub 页面，文件/文件夹前会出现复选框
    3. 勾选你需要的文件，点击下载按钮即可

    其他替代方案

    • DownGit：访问 https://minhaskamal.github.io/DownGit，粘贴仓库中某个文件夹的链接即可单独下载
    • GitHub RAW 直链：单个小文件可直接点击 Raw 按钮保存
